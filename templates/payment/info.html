Payment security


We use Stripe for all our payments. By using Stripe.js and the elements library that Stripe offers, your credit card is never touched by our servers. This technique also makes the payment system PCI compliant.

If you want to read more about the security of Stripe, please consult the following documentation at Stripe.com.

The studlan project is open sourced at GitHub, so you are free to inspect the safety of the code yourself.
